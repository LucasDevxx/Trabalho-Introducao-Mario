# 📝 Sistema de Correção de Provas

Este projeto em Python foi desenvolvido para corrigir automaticamente uma prova com base em um gabarito e nas respostas de um aluno.

---

## 📌 Objetivo

O objetivo do sistema é automatizar o processo de correção de provas, informando:

* Quantidade de acertos
* Quais questões foram erradas
* Se o aluno foi aprovado ou reprovado

---

## 🚀 Como clonar o projeto

No terminal (CMD):

```
git clone https://github.com/LucasDevxx/Trabalho-Introducao-Mario.git
cd TesteMario.py ou Documents
```

---

## ⚙️ Tecnologias utilizadas

* Python
* Lógica de programação
* Estrutura de repetição (`while`)

---

## 📖 Funcionalidades

* Inserção do gabarito da prova (10 questões)
* Inserção das respostas do aluno
* Comparação automática das respostas
* Cálculo de acertos
* Listagem das questões erradas
* Verificação de aprovação (mínimo de 6 acertos)

---

## 🧠 Lógica do programa

O sistema funciona da seguinte forma:

1. O usuário digita o gabarito da prova
2. O usuário digita as respostas do aluno
3. O sistema compara as respostas
4. Conta os acertos
5. Mostra quais questões foram erradas
6. Exibe se o aluno foi aprovado ou reprovado
